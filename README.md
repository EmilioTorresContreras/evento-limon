# Instalaciones que realice

## Clerk

``pnpm add @clerk/nextjs``

Obtén tus claves (Publishable Key y Secret Key) y ponerlas en el .env.local

### Instala Clerk UI

``pnpm add @clerk/themes``

### Instala Clerk Lenguajes

``pnpm add @clerk/localizations``
## Convex

``pnpm add convex``
``pnpm convex dev``

# Para ver si hay algún error: 
``pnpm run lint``
